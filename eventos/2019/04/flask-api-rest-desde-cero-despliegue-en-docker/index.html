Informacion:

Este próximo martes 30 de abril tendremos como invitado especial a Miguel Celis, CEO & co-founder en Heippi, quien nos estará hablando sobre Flask, cómo hacer un API Rest desde cero y desplegarlo con Docker.

⚡Orden del día:

1) Bienvenida 👋
2) Charla: Flask: API Rest desde cero + Despliegue en Docker 🌶🐋
3) Preguntas 💬
4) Hablemos de Comunidad 😊
5) Despedida y comida 🎉🍔

💡Mas detalles: conoceremos aspectos claves sobre Flask, cómo utilizarlo para desarrollo ágil de API Rest y realizar despliegue en un ambiente de producción de manera sencilla gracias Docker. También se expondrá un caso de uso con Autenticación con OAuth2, manejo de roles y CRUD usuarios.
