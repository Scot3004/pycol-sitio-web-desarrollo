Informacion:

¡Hola a todos!

Anunciamos un nuevo Meetup, donde veremos las siguientes charlas:

- Introducción a serverless en AWS con python

Se desarrollará una pequeña aplicación serverless con algunas herramientas que tenemos a disposición.

- Visualización de datos geocodificados con Python y Google Maps

Explicar cómo podemos desarrollar una aplicación para geocodificar y georeferenciar datos de diferentes ubicaciones en un mapa utilizando como motor de bases de datos SQLite, lenguaje de programación Python versión >= 3.0, y la integración con la API actual de Google Maps para la visualización.
