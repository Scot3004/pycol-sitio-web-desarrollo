¡Hola a todos!

Anunciamos un nuevo Meetup, donde veremos las siguientes charlas:

- Estadística para hackers

Ideas centrales en estadística desde un punto de visto simple y computacional (con Python) aprovechando simulación y generación de números aleatorios; y como algunas de estas ideas se relacionan con conceptos de machine learning y métodos Bayesianos.

- OpenWhisk Serverless

Un breve vistazo a OpenWhisk como plataforma cloud Serverless y open source: Conceptos, arquitectura y una demo realizando un API con Python
