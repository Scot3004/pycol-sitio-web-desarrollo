Informacion:

Muchas veces necesitamos conectar sistemas a Django que no solo son páginas, sino comunicar con aplicaciones mobiles, con aplicaciones para televisores, o incluso muchas veces conectarnos desde otros servidores. Para poder hacer esto por mucho tiempo se utilizaron los Web Services a través de SOAP, Pero últimamente la tendencia ha sido a la creación de RESTFull APIs, y por esto hoy aprenderemos a como crear de manera sencilla un API en Django a través de la librería Django Rest Framework.

En esta charla veremos que es un recurso, que es un API y como podemos crear nuestros propios RESTFull APIs en Django de una manera rápida y fácil.

Django Rest Framework la herramienta que te ayudará a comunicar Django con cualquier sistema no solo paginas webs, Presentado por Luis Moncaris Gonzalez, Estudiante de último semestre de Ingeniería de Sistemas de la Universidad de Cartagena, desempeñado en el desarrollo WEB con 2 años de experiencia en Django, actualmente labora como desarrollador backend en Nokia Networks, entusiasta en Django, Node.js y muchas otras tecnologías.
